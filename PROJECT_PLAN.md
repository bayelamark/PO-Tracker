# PO Tracker Project Plan

## Project Description

PO Tracker is a web application that will alow users to search for Pokemon Cards, view prices, and save cards to their collection.

## Main Features

- Search for Pokemon card
- View card details, rarity and prices
- Add cards to a watchlist
- Track price history
- Set price alerts
- Store data using MongoDB
- Make the website mobile-friendly

## Technologies Used

- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MongoDB
