# PO Tracker

PO Tracker is a web application that allows users to search for Pokémon cards, view current card prices, and create a personal watchlist.

## Planned Features

- Search for Pokémon cards by name
- View card images and information
- Display current market prices
- Add cards to a personal watchlist
- Remove cards from the watchlist
- Store user and card information with MongoDB

## Technologies

- React
- Node.js
- Express.js
- MongoDB
- Pokémon TCG API

This project is currently in development as my senior project.
